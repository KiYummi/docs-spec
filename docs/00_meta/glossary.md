## 8. 使用规则

### 8.1 命名规范

- 所有变量、函数、组件命名必须遵循本字典中的术语定义
- 禁止使用本字典未收录的缩写或自创术语
- 发现新术语必须先注册到本字典，再使用

### 8.2 更新流程

1. 发现新术语
2. 提交到字典审核
3. 审核通过后注册到字典
4. 通知所有 Agent

### 8.3 冲突处理

- 同一概念使用相同术语
- 发现冲突以本字典为准
- 冲突由 Coordinator 决策
