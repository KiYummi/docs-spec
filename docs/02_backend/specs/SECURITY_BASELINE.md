# 安全基线 (Security Baseline)

> **doc_type**: SECURITY_BASELINE
> **version**: 1.0
> **用途**: 全平台开发时必须遵循的安全修补基线

---

## 1. 注入防护

| 检查项 | 说明 | 实现方式 |
|--------|------|---------|
| SQL 注入 | 参数化查询 | ORM / Prepared Statement |
| XSS | 输出编码 | HTML/JS/CSS 编码 |
| SSRF | 白名单校验 | URL 白名单 |

## 2. 认证与授权

| 检查项 | 说明 | 实现方式 |
|--------|------|---------|
| 密码安全 | 加密存储 | bcrypt/argon2 |
| Token 安全 | 短期 + 轮换 | JWT + Refresh Token |
| 权限校验 | 每个端点 | RBAC/ABAC |

## 3. 数据安全

| 检查项 | 说明 | 实现方式 |
|--------|------|---------|
| 传输加密 | HTTPS | TLS 1.2+ |
| 存储加密 | 敏感字段 | AES-256 |
| 日志脱敏 | 禁止打印敏感信息 | 正则替换 |

## 4. 多租户隔离

| 检查项 | 说明 | 实现方式 |
|--------|------|---------|
| 数据隔离 | tenant_id | RLS / ORM 拦截器 |
| 权限隔离 | 角色 + 数据范围 | RBAC |

## 5. 限流

| 级别 | 场景 | 默认值 |
|------|------|--------|
| 全局 | 所有请求 | {N} req/min |
| 认证 | 登录/注册 | {N} req/min |
| 写操作 | POST/PUT/DELETE | {N} req/min |
| 敏感操作 | 转账/删除 | {N} req/min |
