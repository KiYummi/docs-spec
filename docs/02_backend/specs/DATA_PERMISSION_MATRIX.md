# 数据权限矩阵 (Data Permission Matrix)

> **doc_type**: DATA_PERMISSION_MATRIX
> **phase**: Phase_3
> **用途**: 定义数据权限控制方案

---

## 1. 权限维度

| 维度 | 说明 | 实现方式 |
|------|------|---------|
| 功能权限 | 能否访问某功能 | RBAC |
| 数据权限 | 能看到哪些数据 | 数据范围 |
| 字段权限 | 能看到哪些字段 | 字段级控制 |

## 2. 数据范围

| 范围 | 说明 | 过滤条件 |
|------|------|---------|
| 全部 | 可看所有数据 | 无限制 |
| 本部门 | 只看本部门数据 | `dept_id = user.dept_id` |
| 本人 | 只看自己的数据 | `owner_id = user.id` |
| 自定义 | 按规则过滤 | 自定义条件 |

## 3. 实现方案

- 数据库层：RLS（Row Level Security）
- ORM 层：全局拦截器自动注入过滤条件
- API 层：响应字段过滤
