# UAT Sign-off 检查清单

UAT 验收与发版前逐条检查，确保所有前置条件和资产就绪。

## 检查项

### UAT 前置条件（6 项）

- [ ] PRD 完整 — 所有功能点已定义且无 TBD
- [ ] ER 已更新 — 全局 ER 图与实际一致
- [ ] 枚举已更新 — 状态机/枚举值已注册
- [ ] LLD/契约已更新 — API 契约与代码一致
- [ ] 测试报告通过 — P0/P1 全修，P2 有计划
- [ ] step_plan 全 Done — 所有任务卡片已完成

### 发版资产验收（9 项）

- [ ] PRD 文档 — 最终版已归档
- [ ] LLD 文档 — 最终版已归档
- [ ] 测试报告 — 含红蓝对抗结果
- [ ] 红蓝对抗报告 — 已归档
- [ ] 缺陷定级报告 — P0~P3 清单已确认
- [ ] 技术债清单 — 已写入 technical_debt.json
- [ ] 种子数据脚本 — 可重复执行
- [ ] 备份策略 — 已验证恢复流程
- [ ] Feature Flag — 租户白名单已配置

### 热修 Sign-off（5 项）

- [ ] PRD 核心 — 修改涉及的功能点已记录
- [ ] ER 同步 — 表变更已注册
- [ ] 枚举同步 — 状态/枚举变更已注册
- [ ] LLD 更新 — 设计文档已同步
- [ ] API 契约同步 — 契约文件已更新
