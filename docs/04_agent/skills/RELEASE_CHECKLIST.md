# 发版检查清单

正式发版前逐条检查，确保版本、安全、数据库、测试、资产全部就绪。

## 检查项

### 版本与资产

- [ ] 版本号一致性 — CHANGELOG 与 Git Tag 一致
- [ ] 资产同步率 — OPENAPI_SPEC.yaml 包含所有新增接口
- [ ] 遗留物扫描 — 无大量未使用组件或冗余变量

### 安全与监控

- [ ] 安全与监控 — 性能/错误阈值告警已配置

### 数据库

- [ ] 迁移风险评估 — 迁移脚本已评估回滚，Staging 验证，Schema Freeze 审批
- [ ] 在线表变更 — 大表 DDL 支持 Online DDL，锁表时间窗口评估，分批变更计划

### 测试与审查

- [ ] 红蓝对抗用例全绿 — 安全测试通过
- [ ] L3 全量集成审查已通过 — 架构审查完成

### 资产就绪

- [ ] 种子数据导入脚本就绪 — 可重复执行
- [ ] Feature Flag 租户白名单就绪 — 灰度可控
- [ ] 缺陷 SLA 达标 — P0 2h / P1 24h
- [ ] 性能基线达标 — 热点查询 <500ms

> 停止打勾，先解决：

- 性能基线未达标 → 需优化后重新测试
