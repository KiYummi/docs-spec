# Hotfix 快速审查清单

Hotfix 模式下快速审查，确保最小化修改和安全底线。

## 检查项

### 修改范围

- [ ] 最小化修改 — 改动范围最小化，无顺手优化
- [ ] 核心功能恢复 — 修复目标功能已恢复

### 安全底线

- [ ] 无 SQL 拼接 — 必须参数化查询
- [ ] 无用户输入直出 — 必须转义
- [ ] 无权限逻辑修改 — 如有修改需人工确认

### 结果确认

- [ ] 无副作用 — 修改不影响其他功能
- [ ] 安全合规 — 通过安全底线检查

## 红旗

> 停止打勾，先解决：

- 修改范围超出 bug 修复 → 缩小范围
- 涉及权限逻辑变更 → 必须人工确认
- 缺少回滚方案 → 先准备回滚
