# 发版前检查清单

> **阶段**: 发版阶段
> **触发时机**: 发版前

---

## 检查项

### 文档齐备

- [ ] API 文档 — API 契约是否更新？
- [ ] 用户手册 — 用户手册是否更新？
- [ ] 部署指南 — 部署指南是否更新？
- [ ] 变更日志 — 变更日志是否记录？

### 代码质量

- [ ] 单元测试通过 — 所有单元测试是否通过？
- [ ] 集成测试通过 — 所有集成测试是否通过？
- [ ] 代码审查通过 — 代码审查是否完成？
- [ ] 安全审查通过 — 安全审查是否完成？

### 数据库

- [ ] 迁移脚本 — 数据库迁移脚本是否准备？
- [ ] 回滚脚本 — 数据库回滚脚本是否准备？
- [ ] 数据备份 — 数据是否备份？

### 部署准备

- [ ] 环境配置 — 生产环境配置是否就绪？
- [ ] 监控告警 — 监控告警是否配置？
- [ ] 回滚方案 — 回滚方案是否准备？

## 红旗

> 停止打勾，先解决：

- 单元测试未通过 → 修复后重新测试
- 安全审查未通过 → 修复安全问题
- 回滚脚本缺失 → 补充回滚脚本
