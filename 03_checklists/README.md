# 质量检查清单目录

> **用途**: 存放各阶段质量检查清单，确保文档和代码质量

---

## 清单列表

| 清单 | 阶段 | 说明 |
|------|------|------|
| prd-checklist.md | 需求阶段 | PRD 质量检查 |
| lld-checklist.md | 设计阶段 | LLD 质量检查 |
| api-checklist.md | 设计阶段 | API 契约检查 |
| code-checklist.md | 编码阶段 | 代码质量检查 |
| test-checklist.md | 测试阶段 | 测试质量检查 |
| release-checklist.md | 发版阶段 | 发版前检查 |

## 使用方式

1. 在对应阶段加载对应清单
2. 逐项检查，打勾确认
3. 未通过项必须修正后重新检查

## 清单规范

### 检查项格式

```
- [ ] 检查项描述 — 检查标准
```

### 严重等级

| 等级 | 说明 | 处理方式 |
|------|------|---------|
| P0 | 阻塞 | 必须修复，否则不能放行 |
| P1 | 重要 | 应该修复，可以延期 |
| P2 | 建议 | 可以不修复，记录技术债 |
