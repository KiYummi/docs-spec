# 发版检查清单

正式发版前逐条检查，确保版本、安全、数据库、测试、资产全部就绪。

## 检查项

### 版本与资产

- [ ] 版本号一致性 — CHANGELOG 与 Git Tag 一致
- [ ] 资产同步率 — API 契约包含所有新增接口
- [ ] 遗留物扫描 — 无大量未使用组件或冗余变量

### 安全与监控

- [ ] 性能/错误阈值告警已配置

### 数据库

- [ ] 迁移风险评估 — 迁移脚本已评估回滚，Staging 验证
- [ ] 在线表变更 — 大表 DDL 支持 Online DDL，锁表时间窗口评估

### 测试与审查

- [ ] 安全测试通过
- [ ] 集成审查已通过

### 资产就绪

- [ ] 种子数据导入脚本就绪 — 可重复执行
- [ ] Feature Flag 租户白名单就绪 — 灰度可控
- [ ] 缺陷 SLA 达标
- [ ] 性能基线达标

## 红旗

> 停止打勾，先解决：

- 迁移脚本无回滚方案 → 阻断发版
- 安全测试未通过 → 安全风险
- 性能基线未达标 → 需优化后重新测试
