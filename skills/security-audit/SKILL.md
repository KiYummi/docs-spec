# 安全审计检查清单

安全专项审计时逐条检查，覆盖注入防护、越权防护、数据安全、凭证与隔离。

## 检查项

### 注入防护

- [ ] SQL 注入 — 参数化查询 / ORM 全覆盖扫描
- [ ] XSS — HTML 过滤 + 前端危险指令禁用
- [ ] RCE — eval() 绝对禁止 + 沙箱隔离

### 越权防护

- [ ] 水平越权 — IDOR 扫描 + 租户隔离
- [ ] 垂直越权 — RBAC + ABAC 端点覆盖

### 数据安全

- [ ] PII 脱敏 — 日志禁止明文
- [ ] 审计日志 — 哈希链连续 + 长期保留
- [ ] 输入校验 — DTO 物理拦截 + 白名单

### 凭证与隔离

- [ ] 凭证安全 — 加密存储 + 连接器凭证保护
- [ ] 多租户隔离 — ORM 自动 tenant_id + 禁止手动 WHERE 拼接

## 红旗

> 停止打勾，先解决：

- 发现 eval() 或字符串拼接 SQL → 立即修复
- PII 字段明文出现在日志 → 先修复脱敏
- 审计日志哈希链断裂 → 数据不可信
