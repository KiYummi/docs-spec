# 文档骨架目录

> **用途**: 存放各种文档的骨架模板，供项目初始化时使用

---

## 使用方式

1. 复制骨架到项目文档目录
2. 替换 `{占位符}` 为实际内容
3. 按需增删章节

## 骨架列表

| 骨架 | 说明 | 对应模板 |
|------|------|---------|
| prd-skeleton.md | PRD 文档骨架 | `docs/08_templates/PRD_Template.md` |
| lld-skeleton.md | LLD 文档骨架 | `docs/08_templates/Sub_LLD_Template.md` |
| api-skeleton.md | API 契约骨架 | `docs/08_templates/API_CONTRACT_Template.md` |
