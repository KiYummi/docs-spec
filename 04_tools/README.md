# 工具目录

> **用途**: 存放文档工程体系的自动化工具脚本骨架

---

## 工具列表

| 工具 | 功能 | 语言 | 说明 |
|------|------|------|------|
| baseline-extractor.py | 原子需求提取 | Python | 从 PRD 中提取原子需求 |
| index-updater.py | 索引更新 | Python | 扫描 @req-id 标签，重建索引 |
| residual-scanner.py | 残差扫描 | Python | 语义覆盖率 + 术语一致性检查 |
| patch-applier.sh | 补丁应用 | Bash | 批量应用结构化补丁 |
| ci-audit.sh | CI 集成 | Bash | 编排上述脚本，输出审计报告 |

## 使用方式

1. 根据项目技术栈选择合适的工具
2. 参考骨架实现具体功能
3. 集成到 CI/CD 流程

## 注意

- 以上工具为骨架（伪代码），不提供可运行实现
- 项目团队根据实际情况实现
- 建议使用项目熟悉的语言和框架
